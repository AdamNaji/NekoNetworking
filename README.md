# Neko Game Engine

Neko is a 3D game engine based on SDL2 and OpenGL ES 3.0 that works on Desktop, WebGL2 and Nintendo Switch (port to Android and iOS possible) used at [SAE Institute Geneva](https://sae.swiss).

## Libraries used
- SDL2
- GLAD
- Box2D
- googletest
- taskflow
- imgui
- sole uuid
- xxhash
- std_image
